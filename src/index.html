Plain & Smooth

This is a go-to tool collection for website design and development that you can access quickly and easily, without having to navigate through bookmark folders repeatedly.

To reduce eye strain, I opted for lower colour tones and a plain style. This concept is what inspired "Plain & Smooth".

  • Visual Studio Code

    The most fully featured code editor with plenty of extensions.

  • Cacher

    Cacher is a code snippet organizer with a clear layout that enables you to search snippets easily and syncs with gist.

  • Pieces

    Pieces is a code snippet manager with generative AI. Quickly import snippets from VS code, browser, and even scan images.

  • CodePen

    Essential tool for testing and showcasing HTML, CSS and JavaScript code snippets. No setup is required.

  • CodeSandBox

    Cloud-based development platform that works with javaScript included libraries such as React, Vue and Angular.

  • Dev Samples

    Dev Samples is a large snippet library that covers HTML, CSS, Git, multiple JS libraries and more.

  • HTML5 Reference | MDN

    The essential reference for entire developers for the dairy basis development.

  • HTML Entity Reference CSS-Tricks

    A helpful HTML entity reference gives you an HTML entity name, numeric code, hex code and ISO code.

  • HTML Entity Lookup

    The useful entity search engine. You can search by regular letters for specific symbols.

  • CSS Reference | MDN

    As with HTML, it’s the essential reference for the dairy basis development for entire developers.

  • CSS Reference | Codrops

    It’s a visually intuitive CSS reference with colourful tags. Easy search, quick solve.

  • Can I use?

    One of the essential tools to check up on supported technologies on desktop and mobile web browsers.

  • 100L5

    CSS generator tools for cubic-bezier, glitch effect, glass morphism and many more creative visual elements.

  • Griddy

    Handy grid pattern generator to make complex layouts. It’s easy to build up your layout.

  • Neumorphism.io

    Awesome neumorphism generator to make smooth and beautiful elements. I used this to design this website.

  • Beautiful CSS box-shadow Examples

    Over 90 patterns of beautiful box-shadow samples. Just click to copy styles for your site design.

  • Smooth Shadow

    Excellent intuitive UI box-shadow generator with a plugin also available on Figma.

  • Simplecss.eu

    Media query generator for hundreds of mobile devices including iPads, iPhones, and Androids.

  • Dart Sass

    The latest recommended version of CSS pre-processor that lets you use Javascript features.

  • BEM

    A systematic CSS naming methodology to organize CSS classes and help you build reusable components.

  • NERDCAVE

    Handy tailwind cheat sheet. Easy search and double-click to copy codes to your projects.

  • JavaScript Reference | MDN

    Another essential reference from MDN for A to Z of JavaScript.

  • JavaScript Event Reference | MDN

    This is the master reference of JavaScript DOM events.

  • You might not need jQuery

    A complete dictionary to translate jQuery to Vanilla JavaScript also can be used as a DOM manipulation reference.

  • Unsplash

    One of the most familiar image resource sites also known as Figma plugin for high-resolution photos.

  • O-DAN

    A powerful search engine for high-quality photo images through multiple world-popular image resource sites.

  • 500px

    A global pro and high-amateur photo-sharing platform with the largest number of users based in Toronto, Canada.

  • rawpixel

    One of the high-quality photo resource sites with thousands of free images and premium collections.

  • nappy

    nappy is a website dedicated to providing beautiful, high-res photos of black and brown people.

  • Burst

    A high-quality free stock photo site powered by Shopify. It has a wide genre of photos stock and is easy to access.

  • ResourceBoy

    The best design resource site providing mockups, textures, effects and more. If you’re a designer, it’s worth checking it.

  • Foodiesfeed

    Foodiesfeed is the best food photo resource site that is completely free to download.

  • Coverr

    Coverr is an excellent free video resource site. The beta version of the stock music service is also available now.

  • Google Fonts

    The most famous and reliable font resource by Google. You might want to check their constant update periodically.

  • Fontshare

    Fontshare is a brand-new free font service run by Indian Type Foundry. You can use pro-grade excellent fonts for free.

  • Font Squirrel

    Font Squirrel is a license-free font resource site which has a wide variety of font styles.

  • MyFonts

    The world's largest online font store, created by Bitstream Inc. The store has a massive stock of fonts you’ve ever seen.

  • iFonts

    iFonts is a great place to discover & download thousands of free fonts for Personal Use.

  • Font Awesome

    One of the popular icon toolkits for the web, based on CSS and Less. It’s easy to use with a useful icon lineup.

  • Line Awesome

    Line Awesome is icon-based outline font created as an alternative to Font Awesome 5.11. 2.

  • LineIcons

    LineIcons is a handcrafted icons library that consist of 8400 + free and premium icons.

  • Icons8

    Icons8 has the world’s largest collection of icons and allows us to recolour. The plugin is available on Figma as well.

  • Meterial Icon

    Well-known icon set which is a part of the Material Design System Google created.

  • Streamline

    The world’s largest icon library with 170,000 systems. The plugin is available on Figma.

  • Coolors

    Coolors is a generative colour palette tool that can quickly create thousands of colour schemes.

  • Color Safe

    Excellent colour evaluation and generator tool that provides you WCAG Guidelines of text and background contrast ratio.

  • Brand Palette

    A corporate identity library that provides corporate colours, typeface and brand logo methodologies.

  • Colormind

    An excellent colour generator powered by AI that can generate colour palettes from pictures, videos and art.

  • Gradient Hunt

    A beautiful gradient gallery that has thousands of different colour gradients you can use or download in png.

  • Awwwards

    You'll see here the best innovative web designs that are recognized and promoted from all over the world.

  • FWA

    Another international award platform. The name stands for “Favorite Website Awards”.

  • Dribbble

    One of the classic galleries to discover many sophisticated designs, especially for web and apps.

  • Pinterest

    This is my favourite design scrap gallery where you can get inspiration from various genres of design. Hit me up there!

  • 1GUU

    Japan-based international web design gallery. Easy to find your inspiration through animated thumbnails.

  • S5-Style

    Another Japan-based web design gallery that curates over 8,000 websites in versatile categories.

    One of the neat web design curations focuses on local Japanese sites.

  • Muuuuu.org

  • Brilliant Design Collect

    The collection on this curated site conceptualized functionality in the most aesthetic proportions.

  • SANKOU!

    Simple and handy local web design gallery. You can bookmark your favourites.

  • LP ARCHIVE

    High-quality landing page design collection from Japanese websites.

  • WAVE

    WAVE is an essential tool for evaluating web accessibility based on Web Content Accessibility Guideline (WCAG).

  • Codrops

    This is my grimoire of the interaction design. If you’re a designer, you must check it.

  • CodePen

    Essential tool for testing and showcasing HTML, CSS and JavaScript code snippets. No setup is required.

  • Modulesss

    Modulesss is a components design gallery collected from Japanese websites.

  • Parts.

    Parts. is a design gallery for specific pages, components, elements, and visual graphics.

  • Landingfolio

    Landingfolio features a landing page design gallery and also lists other pages and components.

  • Codyhouse

    A lightweight framework for building accessible, bespoke interfaces.

  • unDraw

    unDraw is a set of Open-source illustrations for any idea you can imagine and create.

  • humaaans

    Brilliant tool for illustration of people. You can customize by choosing every piece of parts.

  • blush

    This is also a customizable illustration tool. The handy plugin is available on Figma!
